HTML Element Inspector

1. Elements vs. Tags vs. Attributes

Tag: The keywords surrounded by angle brackets (e.g., <a>).

Attribute: Extra information inside the opening tag (e.g., href="link.com").

Element:The entire package: opening tag + attributes + content + closing tag.

2. Tag Types

Paired Tags (Opening and Closing)

These tags wrap around content.

  • <p>This is a paragraph.</p>
  • <strong>Bold text</strong>

Self-closing Tags (Empty Elements)

These tags do not contain text or child elements, so they don't need a closing tag.

  • <br> (Line break)
  • <hr> (Horizontal rule)

3. Attribute Value Types

Attributes can hold different types of data. Here are 5 examples:

  • Text Value: <img alt="A picture of a cat"> (The alt text describes the image).
  • URL Value: <a href="https://upflairs.com"> (Links to a specific web address).
  • Number Value: <textarea rows="5"></textarea> (Defines exactly 5 rows of height).
  • Boolean Value (True/False): <input type="checkbox" checked> (The presence of 'checked' means true).
  • Identifier (ID) Value: <div id="header-section"> (A unique string for CSS/JS targeting).
