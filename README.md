# Upflairs-Day2-Assignment
Day 2 HTML Assignment
This repository contains the completed tasks for the Day 2 HTML assignment. The project demonstrates a fundamental understanding of HTML structure, including the use of various HTML tags, block-level and inline elements, heading hierarchies (h1-h6), and the application of global attributes like id and class. Each task is well-commented to explain the purpose and implementation of the code. This project was built to practice creating organized and semantic web layouts.
